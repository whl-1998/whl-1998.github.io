Vue表单输入绑定

这块内容主要是介绍 v-model 指令，用于表单输入文本与页面文本的双向数据绑定。
比如像这样：

<div id="example">
    <input v-model="message" placeholder="edit me">
    <p> Message is: {{message}} </p>
</div>

<script>
    let example = new Vue({
        el: '#example',
        data: {
            message: ''
        },
        methods: {
        }
    })
</script>

后续的内容也没什么新的，基本上都是数据绑定显示，其余的都是比如 textArea, select, checkbox ... 这类 input 元素的示例。

比较有用的是Vue提供的修饰符：

  • .lazy 这个会在触发 change 事件后在进行双向绑定，比如现在是需要敲回车才会绑定，而不是立即绑定：

    <input v-model.lazy="message" placeholder="edit me">

  • number 可以自动将用户输入类型转为数值类型，但如果是后续追加了无法转换的字符，比如 ‘abc’ 那么还是会按 ‘123’ 转数值。

    <div id="example">
    <label>
        <input v-model.number="message" placeholder="edit me">
    </label>
    <p> Message is: {{message}} , and the type is {{type}} </p>
</div>

<script>
    let vm = new Vue({
        el: '#example',
        data: {
            message: '',
            type: 'none',
        },
        watch: {
            // 如果 message 发生改变，那么这个函数需要执行
            message: function () {
                this.type = typeof this.message
            }
        },
    })
</script>

  • trim 可以自动去除首尾的空字符

    <input v-model.trim="message" placeholder="edit me">
