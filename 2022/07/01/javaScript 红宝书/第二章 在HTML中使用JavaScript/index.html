第二章 在HTML中使用JavaScript

title: 第二章 在HTML中使用JavaScript
date: 2022-07-01 15:03:27
tags: JavaScript红宝书阅读日志
categories: JavaScript, 读书笔记

一开始介绍了怎么把 JavaScript 放到网页中执行，即在也HTML页面中通过 <script> 标签引入 JavaScript 代码的方法，这没什么好说的。
值得关注的是：不要在代码部分的任何地方出现 </scirpt> ，以下是错误示范

<script type="text/javascript"> 
    function sayScript() { 
        alert("</script>"); 
    } 
</script>

以上代码并不会成功执行，而是会报错。
若一定要在代码部分出现 </scirpt> ，请使用转义字符，示例如下：

<script type="text/javascript"> 
    function sayScript() { 
        alert("<\/script>"); 
    } 
</script>

若是采用 src 属性引入外部 script 文件，并且还在标签内嵌入了代码，那么嵌入的代码将被忽略。此时仅仅会下载并执行外部脚本文件。

<script type="text/javascript" src="http://www.somewhere.com/afile.js">
    function sayScript() { 
        alert("</script>"); 
    } 
</script>

并且执行的顺序是依次的，换句话说，在第一个<script>元素包含的代码解析完成后，第二个<script>包含的代码才会被解析，然后才是第三个、第四个…

引入外部 script 时，引入代码应该放在 body 尾部而不是 head 中，这样能够在引入较多外部文件的情况下，加快页面呈现的速度：

<!DOCTYPE html> 
<html> 
    <head>
        <title>Example HTML Page</title> 
    </head> 
    <body> 
        <!-- 这里放内容 --> 
         <script type="text/javascript" src="example1.js"></script> 
        <script type="text/javascript" src="example2.js"></script> 
    </body> 
</html>
